{% header h2 | Using the Spot Illustration Helper %}

The SVG Razor helper function should be used for all system illustrations in lieu of SVG sprite sheets or raster image assets. This ensures a single source of truth for all illustrations. The SVG helper is invoked by @Svg. and then adding the illustration name, like so:

{% highlight html linenos %}
@Svg.Spot.Location
{% endhighlight %}

By default, every spot illustration will inherit the text color of its parent.

{% highlight html linenos %}
@Svg.Spot.Location.With("fc-orange-500")
{% endhighlight %}

Our helper supports arbitrary classes as well.

{% highlight html linenos %}
@Svg.Spot.Location.With("fc-orange-500 js-dropdown-target another-class-name")
{% endhighlight %}
{% icon Alert %}
Note: Our @Svg helper isn’t available within our documentation. You can call SVGs using our custom Spots liquid tag: {% raw %}{% spot Location | optional additional classes | pixel dimension %}{% endraw %}
